Canvas

¡Dibuja con comandos! Canvas es un elemento de HTML, el cual permite dibujar gráficos mediante secuencias de comandos (comunmente JavaScript).

Read More

¿Qué puedo hacer con canvas?

Este es uno de los ejemplos de lo que es posible hacer con canvas, como se puede observar en la parte de abajo hay dos triangulos uno esta dibujado estaticamente mientras que el otro gira en su propio eje gracias a comandos de animacion, es posible hacer muchas figuras y dibujos, hasta poder llegar a hacer dibujos complejos como lo pueden ser paisajes animados, o personajes de television o caricaturas, canvas por medio de Java Script ofrece varias herramientas de trazado y dibujo, con ello se lleva acabo estos tipos de animaciones y trazados.

¿Como se hace?
