Canvas

¡Dibuja con comandos! Canvas es un elemento de HTML, el cual permite dibujar gráficos mediante secuencias de comandos (comunmente JavaScript).

Read More

Dinámico

Canvas puede añadir interacción con el usuraio con JavaScript, puedes interactuar con el teclado, el ratón o lo que tú desees.

Multipropósito

Con canvas tu imaginación es tu límite, crea gráficos, animaciones, juegos, imagenes estáticas o interactivas, tú eliges.

Fácil de utilizar

No necistas muchos conocimientos para empezar a utilizar canvas, en cuestión de minutos tedrás tus propias creaciones.

¿Qué puedo hacer con canvas?

Existen multiples aplicaciones y utilidades para canvas, puesdes crear maravillosos paisajes o retratos a base de código, sin embargo, si lo tuyo son las matemáticas y las estadísticas, puedes utilizar canvas para representar gráficos matemáticos o estadísticos en tiempo real; por otro lado, si las físicas se te dan bien o quieres poner a prueba tus conocimientos puesdes crear un simulador, así comprobar cómo afectaría al cambiar ciertos parámetros; sin embargo, si estos ejemplos no cubren tus necesidades y deseas algo más, puedes crear videojuegos, dependiendo de lo que necesites, puedes crear personajes importando y recortando imágenes para utilizarlos como sprites.

Claro que no, solo te hemos presentado algunos ejemplos, puedes utilizar canvas para hacer elementos decorativos o incluso tu propio logo,las pósibilidades son infinitas, quien sabe, si te lo propones quizás puedes encontrar la forma de hacer algo que no muchos se hayan imaginado.
Algunas limitaciones que posee canvas es no ser responsive, si elijes un tamaño en específico para trabajar deberás de tomarlo en cuenta; si tienes demaciadas funciones, demaciadas intacias y/o demaciadas iteraciones tendrás muchos problemas a la hora de ejecutar los comando, lo que puede realentizar tu computadora o tener fallos con tu navegador.
