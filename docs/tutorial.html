Canvas

¡Dibuja con comandos! Canvas es un elemento de HTML, el cual permite dibujar gráficos mediante secuencias de comandos (comunmente JavaScript).

Read More

¿Qué puedo hacer con canvas?

Este es uno de los ejemplos de lo que es posible hacer con canvas, como se puede observar en la parte de abajo hay dos triángulos uno esta dibujado estáticamente mientras que el otro gira en su propio eje gracias a comandos de animación, es posible hacer muchas figuras y dibujos, hasta poder llegar a hacer dibujos complejos como lo pueden ser paisajes animados, o personajes de televisión o caricaturas, canvas por medio de Java Script ofrece varias herramientas de trazado y dibujo, con ello se lleva a cabo estos tipos de animaciones y trazados.

¿Cómo se hace?

Como se ha mencionado en el párrafo anterior, gracias la etiqueta canvas en HTML y Java Script se puede hacer este tipo de dibujos y es una serie de códigos extensos que tendría que verse paso a paso para comprender al 100%, pero se dará una breve explicación.

- Para comenzar se crea la etiqueta CANVAS en HTML y se define el tamaño de largo y ancho también un ID para reconocerlo

- se procede a crear el dócumento js(Java Script)

DENTRO DE JS (figura verde)

- Se vincula Java Script y se crea una función donde se dibujara por medio de trazados

- Para comenzar a crear un trazado se define la función que será la que reconozca HTML al compilar en este caso "Init()"

- Luego se crea dos variables, una para el que contendrá el canvas y otra para el contexto del canvas y el tipo de dibujo, que será 2D

- A continuación se comienza a trabajar con los trazos y para ello se llama a la variable context y se usa la función beginPath para comenzar el nuevo trazo

- Se llama a context y a la función moveTo para dar coordenadas de que posición principal debe tener ese trazado

- Luego se llama a la variable contexto y se llama a la función LineTo que define un trazado este con coordenadas X, Y con parámetros de 200 y 350

- Se vuelve a llamar a la variable contexto y la función de trazado

- Se llama a contexto y se llama a la funcion fillStyle que dará un estilo de relleno de color verde claro

- Por último se llama a la variable contexto y a la vez se llama a la función que dará el relleno (fill).


(Figura celeste)

- Para comenzar a crear la figura celeste se crean dos variables una que tendrá el ID canvas y la otra variable que tendrá el contexto

- Dentro de este canvas se llama a la función que contiene la figura verde para que la dibuje, de otra manera es imposible visualizarla

- Se llama a la variable ctx y se borra todas las rectas del canvas o trazados para que se re-dibujen

- Se llama a la variable ctx y se llama a la función beginPath para comenzar a crear trazados

- Se llama a la variable ctx y se llama a la función moveTo para comenzar a dibujar desde un punto específico

- Se llama a la variable ctx y se llama a la función lineTo para crear un trazado con las coordenadas x, Y. Y se hace lo mismo con el siguiente LineTo

- Se llama a la variable ctx y luego a la función translate para que pueda girar sobre su propio eje

- Se llama la función rotate con este será posible dar la animación de rotación, este esta convertido de radianes a grados

- Luego se da un relleno de estilo con la función fillStyle color azul claro

- Para rellenarlo se llama a la función fill y se rellena desde el trazado inicial al final

- por último se llama al método windowns y a la función requestAnimationFrame que será una animación y este necesita una función y para ello se le manda por parámetro a draw y se genera un bucle
